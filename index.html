HTML

  • The head element contains information of about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element defines the metadata such as the character set,descroption and viewport.
  • Elements nested inside the other elements are called the child elements.
  • The element that holds all the children is called as parent element.
  • An element refers to the entire element from opening tag to closing tag.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • External CSS file will be linked to the HTML file using the HTML link element.
  • Inline CSS, internal CSS style and external style sheet are the three ways of CSS used to style a webpage.
  • Inline styles apply to a specific HTML tag for a quick and eay changes.
  • Internal style sheet contains the CSS rules for the webpage in head section of the HTML file.
  • A CSS rule contains a selector and a declaration block
  • A declaration block contains multiple declarations.
  • Declaration contains two components: the CSS property and the value of the property.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull: fetches the latest code from the remote branch
  • git push: pushes the commits to the remote branch
  • git commit -m: commits the changes with a messange
  • git add -A:staging all the modified files for commit
  • git stash:saving the files aside for later use
